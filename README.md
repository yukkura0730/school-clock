# 学校用にカスタマイズされた時計・タイマー V4.1.2
**機能**  
機能としては下の5つです
1. チャイム着席タイマー
2. タイマー機能（ポロモードタイマー機能は削除しました）
3. UDフォント採用
4. 爆発機能（動画再生機能）
5. ダークモード対応

**導入**  
主に導入方法は2種類あり、1つめはweb版で使う方法でこれはダウンロードなどは不要です。  
2つめはローカルで動かす方法です。これは学校などの制限がある環境で使うことをおすすめします。  
1つめの方法は[このリンク](https://yukkura0730.github.io/school-clock/)をクリックして、タイマーを起動するボタンを押すと起動できます。  
2つめの方法はこのサイトからtimer.htmlをダウンロードして、ファイルをブラウザで開くと起動できます。  
注意　2つめの方法はバージョンアップされても自動更新されません

**使い方**  
タイマーなどの設定方法はタイマーを起動して右上にある設定ボタンを押すと設定画面に移動します。
1日のスケジュールなどはデフォルトがあるのでそれを参考にして設定してください。  
爆発機能（動画再生機能）は動画ファイルや音声ファイルを読み込み（アップロードはされません）をしてチェックを入れて保存をしてから利用してください。  
タイマー機能の音は爆発機能のチェックを外さないと音がなりません
